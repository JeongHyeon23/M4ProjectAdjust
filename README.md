# [Unity 2D] Mobile RPG Game PortFolio
## 1.소개
1. Tcp서버통신과 유니티 엔진기반의 모바일 멀티 RPG 게임에서 몬스터와의 전투 및 2명의 플레이어간 PK를 구현한 게임입니다..
2. Reopsitory에는 유료에셋 사용으로 서버 및 클라의 소스코드만 등록되어 있습니다.
3. 개발기간: 2025.08.29 ~ 2025.12.01 (약 3개월)

   
## 2.개발환경
1. 유니티 2022.3.54f1 LTS
2. C# 서버
3. 구글 프로토 버퍼 사용

## 3.사용기술
1. 어드레서블
   + 어드레서블 시스템을 사용하여 네트워크에 접속전에 모든 데이터시트 및 리소스를 Dictionary로 캐싱하여 관리함
2. FSM 이중 사용 관리
   + FSM을 조이스틱 상태와 플레이어 상태로 이중관리하여 조이스틱에 따른 플레이어의 로직 간결화 및 직관성 강화
3. 서버의 객체 이동 연산 부담 완화 
   + 서버가 스킬과 이펙트 및 투사체를 같은 공간에 입장만 시키고 화면상 보이는 이동은 클라가 담당하여 서버의 객체이동 연산의 부담을 완화
